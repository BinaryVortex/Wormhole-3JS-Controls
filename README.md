# Wormhole 3JS Controls

An interactive 3D wormhole visualization built with Three.js, featuring customizable controls for an immersive space travel experience.

## 🌌 Live Demo

Experience the wormhole live: [https://binaryvortex.github.io/Wormhole-3JS-Controls/](https://binaryvortex.github.io/Wormhole-3JS-Controls/)

## ✨ Features

- **Interactive 3D Wormhole**: Navigate through a dynamic tunnel of glowing rings
- **Particle Effects**: Thousands of particles flowing through the wormhole
- **Customizable Controls**:
  - Speed adjustment
  - Particle count (500-3000)
  - Rotation speed
  - Tunnel and particle colors
- **Visual Presets**: Quick theme switching (Cyber, Sunset, Matrix, Purple)
- **Mouse Navigation**: Move your mouse to steer through the wormhole
- **Responsive Design**: Works on desktop and mobile devices
- **Background Stars**: Immersive starfield for depth

## 🎮 How to Use

1. Clone or download the repository
2. Open `index.html` in any modern web browser
3. Use the controls panel to customize your experience:
   - Adjust sliders for speed, particle count, and rotation
   - Pick colors for the tunnel and particles
   - Try different presets for instant themes
4. Move your mouse to navigate the camera
5. Toggle controls visibility with the button in the bottom-right

## 🛠️ Technologies Used

- **Three.js**: 3D graphics and rendering
- **HTML5**: Structure and canvas
- **CSS3**: Styling and responsive design
- **Vanilla JavaScript**: Logic and interactivity
